SphereMetrics was created as a user-friendly Shiny app with a straightforward interface designed to streamline the process of measuring the area and eccentricity of spheroids. It allows the upload and automated detection of spheroids using JPEG or TIFF file formats and generates robust and objective area and eccentricity measurements. SphereMetrics allows automated spheroid measurement by beginners and does not require advanced coding skills. After running the provided code, users can open the app in the browser and measure spheroids by adjusting settings within the shiny app environment without any coding.
